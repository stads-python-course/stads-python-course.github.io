5: Software Engineers

Task

Read in the csv file on US software engineers, analyze the data, and visualize your findings. Answer the following questions:

  1. What are the top 3 most sought-after job titles?
  2. Compare the average company rating of remote vs. non-remote jobs.
  3. Do full-time remote jobs offer higher salaries than full-time non-remote jobs? (advanced, optional)

You may need to clean the data first. Feel free to use data.csv since it has only the relevant information for this project. But if you want to explore more, you can use full_data.csv.

About the Data

This dataset contains job listings for software engineers in the US, including various attributes such as job title, company, salary, job type, location, and remote work options. The data includes information on the number of hires needed, company ratings, job urgency, and whether the position is remote or not. This data provides insights into job market trends, remote work, and salary comparisons within the software engineering sector.
