4: Programming Languages

Task

In this project, we analyze the popularity of various programming languages over time using data from Stack Overflow. The dataset includes the number of posts per month for different programming languages, starting from September 2008. We'll explore this data using Pandas for data manipulation and Matplotlib for visualizing trends.

Key Tasks:

  1. Data Exploration
  2. Data Cleaning
  3. Data Manipulation
  4. Data Visualization
