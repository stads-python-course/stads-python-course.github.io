6: Google Trends

Task

In this project, we will analyze the relationship between Google search trends and other data sources, such as stock prices and unemployment rates. Specifically, we will explore if search volume for terms like "Unemployment Benefits" can provide insights into actual unemployment rates or if search volume correlates with the price of assets like Bitcoin and Tesla stock.

Key Tasks:

  1. Data Collection and Exploration
  2. Data Cleaning
  3. Data Analysis
  4. Data Visualization
