Welcome!

This website contains all relevant information for the Python course offered by STADS (Students Association for Data Analysis and Statistics) and the University of Mannheim.

We will have eight course sessions from 24.09.-12.11. in SN 163 Manfred Lautenschläger Hörsaal. An additional Q&A Session will be held on 19.11. Finally, the exam will take place on 26.11.

Before the first session, please make sure you completed the setup.

All materials will be provided in English, as is typical in the data science world. We will decide in the first session whether the sessions themselves will be held in English or German. In any case, you can contact me in either language.

This course mainly consist of two parts:

  • Introduction to the topic by working through the w3schools Python Tutorial. The interactive exercises allow us to learn Python efficiently, while focusing directly on its application.
  • Hands-on coding projects to reinforce and deepen your knowledge of the fundamentals you have learned. You will be able to directly apply and combine newly learned material to complete mini-projects provided on the GitHub site. The proportion of hands-on projects will increase over time.

Setup

  1. Python. We will be using Python 3.12 for this course. There are many ways to install Python. If you haven't done so yet, you could use Miniconda.
  2. VS Code. We will be working a lot with Jupyter Notebooks. You can use Jupyter Notebooks in your browser, but it is recommended to use Visual Studio Code. Make sure to install the Python and Jupyter Extensions in VS Code.
  3. Git. All code will be provided in a GitHub repository
    https://github.com/mograev/stads-python-course.git
    If you haven't already done so yet, you can install Git as described below. Alternatively, you may find it easier to use Github Desktop.
    • Windows: Download Git for Windows or just type in your admin powershell
      choco install git
    • Mac: Download Git for Mac or just type in your terminal
      brew install git
    If you are not familiar with Git, you can find a very short guide here.
